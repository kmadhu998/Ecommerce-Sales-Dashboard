# 📊 E-commerce Sales Dashboard

## 📌 Project Overview
This project presents an **interactive E-commerce Sales Dashboard** built using **Power BI**.  
The dashboard provides insights into sales performance, profit trends, product performance, and regional sales distribution.

The goal of this project is to analyze e-commerce sales data and create a **visual business intelligence report** that helps stakeholders make data-driven decisions.

---

## 📂 Dataset
The dataset used in this project contains information about:

- Order ID
- Order Date
- Product Name
- Category
- Sub-Category
- Region
- Segment
- Sales
- Quantity
- Profit

The data represents sales transactions from an e-commerce business.

---

## 🛠️ Tools & Technologies Used

- **Power BI Desktop**
- Data Visualization
- Business Intelligence
- Data Analysis

---

## 📊 Dashboard Features

The dashboard includes the following key metrics and visualizations:

### 🔹 Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Total Quantity Sold
- Total Orders
- Profit Margin

### 🔹 Visual Insights

- Sales by Category
- Profit by Category
- Top Selling Products
- Sales by Region
- Monthly Sales Trend
- Sales by Customer Segment

---

## 📈 Key Insights

Some important insights obtained from the dashboard include:

- The **Technology category generates the highest profit** among all product categories.
- The **West region contributes the largest share of sales**.
- **Consumer segment dominates overall sales** compared to other segments.
- Certain products contribute significantly to total revenue.

These insights help businesses identify profitable categories and high-performing regions.
